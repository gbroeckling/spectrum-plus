# Code of Conduct

## Our pledge

Spectrum Plus is a welcoming project for everyone, regardless of experience
level, background, or coding methods.

## Expected behavior

- Be kind and constructive in issues, PRs, and discussions.
- Assume good intent.
- Give helpful feedback.
- Respect different hardware setups, skill levels, and approaches.

## Unacceptable behavior

- Personal attacks, insults, or dismissive language.
- Harassment of any kind.
- Posting others' private information without consent.
- Trolling or deliberately disruptive behavior.

## Enforcement

The maintainer may remove comments, close issues, or block users who violate
this code. For serious concerns, email the maintainer directly.

## Scope

This code applies to all project spaces: GitHub issues, pull requests,
discussions, and any related communication channels.
